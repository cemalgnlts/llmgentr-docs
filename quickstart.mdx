## 3 Aşamada İlk İsteğinizi Göndermeye Hazılranın

API anahtarınızı alın, API URL'sini tanıyın ve modelinizi seçin.

### Adım 1: API Anahtarınızı alın

- Panelinizi açın: https://llm.gen.tr/#api-keys
- Varsayılan _(default)_ veya yeni oluşturacağınız anahtarı kopyalayın.
- Anahtarınız bu şekilde başlar: `llm-sk-***`

### Adım 2: API URL'si

Ana API URL'si bu şekilde: https://llm.gen.tr/v1 bazı araçlar **v1**'i kendi ekler. Eğer 404 gibi bir hata alırsanız **v1** otomatik eklendiği içindir kaldırabilirsiniz.

### Adım 3: Model Seçimi

Model seçerken aklınızda bir model yok ise en düşük modelden en gelişmiş modele geçiş şeklinde başlayabilirsiniz.

Aşağıdaki modeller genel amaçlıdır ve başlangıç yapmak için iyi bir seçim.

- gemini-2.5-flash-lite
- gpt-5-nano
