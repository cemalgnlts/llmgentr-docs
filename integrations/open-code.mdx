---
title: "OpenCode"
icon: "code"
description: "OpenCode LLM'yi sağlayıcı olarak ekleme."
---

<Steps>
  <Step title="OpenCode indirin">
    İndirme işlemlerini kendi sayfasından yapabilirisiniz: [https://opencode.ai/tr/download](https://opencode.ai/tr/download)
  </Step>
  <Step title="Konfigürasyonu Yapın">
    1. Ev dizininizde bulunan konfigürasyon dosyasını açın: `~/.config/opencode/opencode.json`
    2. Dosyayı aşağıdaki şekilde güncelleyin.
       ```json
       {
         "$schema": "https://opencode.ai/config.json",
         "default_agent": "plan",
         "provider": {
           "llm": {
             "name": "LLM",
             "npm": "@ai-sdk/openai-compatible",
             "models": {
               "glm-5:code": {
                 "name": "GLM 5",
                 "limit": {
                   "context": 202752
                 }
               },
               "glm-5.1:code": {
                 "name": "GLM 5.1",
                 "limit": {
                   "context": 202752
                 }
               },
               "glm-5.1-precision:code": {
                 "name": "GLM 5.1 Precision",
                 "limit": {
                   "context": 202752
                 }
               },
               "minimax-m2.5:code": {
                 "name": "MiniMax M2.5",
                 "limit": {
                   "context": 204800
                 }
               },
               "kimi-k2.5:code": {
                 "name": "Kimi K-2.5",
                 "limit": {
                   "context": 262144
                 }
               },
               "qwen3.5-397b-a17b:code": {
                 "name": "Qwen3.5",
                 "limit": {
                   "context": 262144
                 }
               }
             },
             "options": {
               "baseURL": "https://llm.gen.tr/v1"
             }
           }
         }
       }
       ```
  </Step>
  <Step title="Bağlantı kurma">
    Proje klasörünüzde OpenCode'u çalıştırın.

    ```text
    opencode
    ```

    Daha sonra llm'yi bağlamak için aşağıdaki komutu çalıştırın.

    ```text
    /connect
    ```

    Adımları tamamlayıp bağlantıyı kurun:

    1. Açılan listede LLM seçeneğini arayın (muhtemelen listenin en altında).
    2. LLM seçenğine gelince <kbd>enter</kbd> tuşuna basın.
    3. [API anahtarınızı](https://llm.gen.tr/dash/#api-keys) yapıştırıp kaydedin.
  </Step>
  <Step title="Model seçimi">
    1. OpenCodu başlatın.
    2. `/models` yazıp <kbd>enter</kbd>'e basın.
    3. Açılan listede LLM yazan modellerden birini seçin.
  </Step>
</Steps>
