---
title: "Claude Code"
icon: "code"
description: "LLM modellerini Claude Code ile kullanın."
---

<Steps>
  <Step title="Claude Code kurulumunu yapın.">
    Kurulum için resmi dökümantasyonu takip edin: [https://code.claude.com/docs/en/overview](https://code.claude.com/docs/en/overview)
  </Step>
  <Step title="API Bilgilerini ekleyin.">
    Ev klasörünüzde bulunan `~/.claude/settings.json` dosyasını açın ve içeriği bu şekilde güncelleyin:

    ```json
    {
      "env": {
        "ANTHROPIC_AUTH_TOKEN": "sk-llm-...",
        "ANTHROPIC_BASE_URL": "https://llm.gen.tr",
        "API_TIMEOUT_MS": "90000"
      },
      "model": "glm-5:code"
    }
    ```
  </Step>
  <Step title="Claude Code başlatın.">
    Proje klasörünüze geçin Claude Code'u başlatın:c

    ```shellscript
    claude
    ```
  </Step>
  <Step title="Test edin.">
    Aşağıdaki komutu çalıştırarak eklediğimiz bilgilerin olduğunda emin olun.

    ```text
    /status
    ```

    Bu komuttan sonra model kısmında `glm-5:code` yazdığını görürsek kurulum tamamlanmıştır. Modeli değiştirmek için `/model model-id:code`komutunu kullanabilirsiniz.
  </Step>
</Steps>
