## Genel Kullanım

Aşağıda kendi programınıza eklemeniz için gereken bilgiler yer alıyor

1. `https://llm.gen.tr/v1/chat/completions` adresine `POST` isteği göndermelisiniz.
2. `Authorization` başlığına API anahtarınızı ekleyin.
3. `Content-Type` başlığını `application/json` olarak ayarlayın.
4. **Body** içinde JSON isteiğinizi gönderin.

```http
POST https://llm.gen.tr/v1/chat/completions
Authorization: Bearer API_ANAHTRINIZ
Content-Type: application/json

{
	"model": "gemini-2.5-flash-lite",
	"messages": [
		{ "role": "user", "content": "Merhaba" }
	]
}
```

## Metin Tamamlama

Aşağıda birkaç yazılım dili için örnek kullanım vardır.

<CodeGroup>

```shellscript Curl expandable lines
curl https://llm.gen.tr/v1/chat/completions \
  --request POST \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json' \
  -H 'Authorization: Bearer YOUR_SECRET_TOKEN' \
  --data '{
  "model": "gemini-2.5-flash-lite",
  "messages": [
    {
      "role": "user",
      "content": "Merhaba"
    }
  ]
}'
```


```javascript JavaScript expandable lines
fetch('https://llm.gen.tr/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_SECRET_TOKEN'
  },
  body: JSON.stringify({
    model: 'gemini-2.5-flash-lite',
    messages: [
      {
        role: 'user',
        content: 'Merhaba'
      }
    ]
  })
})
```


```python Python expandable lines
requests.post("https://llm.gen.tr/v1/chat/completions",
    headers={
      "Content-Type": "application/json",
      "Accept": "application/json",
      "Authorization": "Bearer YOUR_SECRET_TOKEN"
    },
    json={
      "model": "gemini-2.5-flash-lite",
      "messages": [
        {
          "role": "user",
          "content": "Merhaba"
        }
      ]
    }
)
```


```php PHP expandable lines
$ch = curl_init("https://llm.gen.tr/v1/chat/completions");

curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, ['Content-Type: application/json', 'Accept: application/json', 'Authorization: Bearer YOUR_SECRET_TOKEN']);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
  'model' => 'gemini-2.5-flash-lite',
  'messages' => [
    [
      'role' => 'user',
      'content' => 'Merhaba'
    ]
  ]
]));

curl_exec($ch);

curl_close($ch);
```
