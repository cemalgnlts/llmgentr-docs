## OpenClaw'ı Yeniden Başlatın

Tarayıcıdan UI kullanarakta tekrar başlatabilirsiniz.

```shellscript
openclaw gateway config.apply --file ~/.openclaw/openclaw.json
```

## Model Seçimi

1. `/models` yazıp llm seçeneğini gördüğünüze emin olun.
2. Daha sonra` /models llm` yazıp model listesini görebilirsiniz.
3. `/model [model adı]` yazıp bir modele geçiş yapabilirsiniz.

<Warning>
  [Kodlama aboneliği](/kodlama-plani) ile kodlama modellerinden birini kullanmak yüksek token maliyetinin önüne geçebilir.
</Warning>
